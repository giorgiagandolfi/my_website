Core Software Infrastructure

The SCOUT framework integrates simulation engines, bioinformatics workflows, and reproducible computational environments to model tumour evolution, process genomic data, and support large-scale cohort generation.

ProCESS

ProCESS is an R-based simulation framework designed for modelling spatial and temporal tumour evolution dynamics. It provides a scalable and reproducible environment for studying cancer growth processes and evolutionary trajectories.

Access the complete documentation and usage guide here .

SCOUT

SCOUT is an R package containing the complete collection of results, analyses, and derived resources presented throughout the study. The package centralises the computational outputs of the project, enabling downstream exploration, reproducibility, and data integration.

nf-core/tumourevo

nf-core/tumourevo is a next-generation bioinformatics workflow developed to reconstruct tumour evolution from whole-genome sequencing (WGS) data.

The pipeline performs advanced downstream analyses of variant and copy-number alterations from tumour-normal matched sequencing assays, supporting both single-sample and multi-region longitudinal studies across independent patient cohorts.

Reproducibility & Computational Environments

To ensure transparent and reproducible research, the SCOUT pipeline is distributed with containerised environments and openly available reproduction scripts.

Containerised Environments

Docker is used to encapsulate the complete SCOUT computational stack, providing consistent software environments across local systems, cloud infrastructures, and HPC platforms.
